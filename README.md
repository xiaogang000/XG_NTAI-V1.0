# XG拟态

>  公众号 : XG小刚

一键免杀冰蝎、哥斯拉等webshell的php、jsp木马文件。(注：不支持流量混淆)



## 使用方法

目前搭载6种php模板、1种jsp模板

```
java -jar XG_NTAI.jar
```

1、选择PHP、JSP页面，选择免杀模板，加密密钥可自定义

2、源码区替换为冰蝎、哥斯拉的php、jsp源码

4、点击免杀，获取免杀后代码即可

5、部分免杀马在第三文本框有特殊传参要求
